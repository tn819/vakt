What vakt does

🔐

Secrets in keychain

No API keys in JSON files. Secrets resolve from macOS Keychain, Linux pass, or environment at sync time.

🛡️

Runtime policy

Define what each MCP server can do. Block dangerous tools before they run. Audit every call.

📡

Multi-provider sync

One command writes config to Claude, Cursor, Gemini, Codex, OpenCode, and Windsurf simultaneously.

☁️

Cloud sandboxes

Route MCP servers to E2B cloud sandboxes. Same config, isolated runtime, no local attack surface.

🔄

Model router

Route completions between local and frontier LLMs. Save costs without changing your workflow.

📊

Audit & observability

SQLite audit log with SIEM export. OpenTelemetry spans for distributed tracing. Compliance-ready.
